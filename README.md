# AI Video Studio

A short-video planning and manifest tool adapted from the workflow schema in [harry0703/MoneyPrinterTurbo](https://github.com/harry0703/MoneyPrinterTurbo).

This v0.1 release creates storyboards and a MoneyPrinterTurbo-compatible starting manifest. It intentionally does not render video. Rendering requires an isolated Python/FFmpeg job runtime, provider credentials, storage lifecycle controls, and media-rights review.

See [SECURITY.md](SECURITY.md) before adding uploads, provider credentials, or rendering.
